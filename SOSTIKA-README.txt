========================================================================
  SOSTIKA - Portable app (with the fixed messaging system + all features)
========================================================================

WHAT THIS IS
  A ready-to-run copy of SOSTIKA. No installer needed. It already contains
  every feature you use, INCLUDING the fixed "message a specific person on
  WhatsApp / Instagram / Facebook" system (search-first, verified chat,
  anti-wrong-box guards).

HOW YOUR FRIEND RUNS IT
  1. Unzip this file anywhere (e.g. Desktop\SOSTIKA).
  2. Open the unzipped folder and double-click  Sostika.exe
  3. First launch: enter your own FREE Google Gemini API key
     (get one at https://aistudio.google.com/apikey ).
  4. For messaging to work, be logged into WhatsApp/Instagram/Facebook
     in your normal default browser first.

  Windows SmartScreen may say "unknown publisher" (the app isn't
  code-signed). Click "More info" -> "Run anyway". This is normal.

PRIVACY
  This copy contains NO personal data - no API key, no memories, no chat
  history. Everyone who runs it starts fresh with their own key.

NOTE
  This is the portable form (run from a folder). A one-click Windows
  INSTALLER (.exe) could not be rebuilt in this environment because the
  build tools (electron-builder) can't be downloaded here. This portable
  version has the identical features and works the same once running.
