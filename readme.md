 # to setup the ai assitant in your personal device 
 follow the following steps 


  1. download the zip file
  2. extract the files from the donwnloaded zip file .
  3. inside that u will get an .exe file / application named sostika . double click it or install it
  4. after the installation , open it
  5. it may ask for mic permission , give it
  6. after it opens . a pop up screen asking for gemini api key will come
  7. then generate the free api key from google ai studio ( https://aistudio.google.com/api-keys )
  8.  put the generated api key into the pop up panel of the application .
  9.  then you are ready to go . start assistant and enjoy
